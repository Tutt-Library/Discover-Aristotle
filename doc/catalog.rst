:mod:`aristotle.catalog` -- Django application for Discovery and Access to Solr Indicies
========================================================================================

Overview
^^^^^^^^

The Django catalog application creates a basic facet view of Solr index of MARC and other
types of bibliographic MODS records from a Fedora Repository.
