:mod:`aristotle.grx` -- Gold Rush Micro-services Django application
===================================================================

Overview
^^^^^^^^
The Gold Rush Micro-services Django application provides an user
interface that queries our open-url providor, `Gold Rush <http://grweb.coalliance.org/>`_,
and then a specific Solr core to generate a mash-up for inclusion in our 
library's website and other web applications.
