What is a discovery service?

THE MESSAGE The Tutt Library's new search interface is an example of a catalog with search facets and other features to help you find and organize the print and electronic resources available through the library. These new types of online catalogs are collectively called next generation discovery layers.

The new TIGER is an open-source project and you can contribute to its design and functionality. The source code repository is available on Github.
