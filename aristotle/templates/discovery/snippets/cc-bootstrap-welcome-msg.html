Welcome to Tutt Library's TIGER Catalog

The new TIGER catalog, also called a Discovery Layer, includes a number of new features like:

  • Faceted search options to narrow search results
  • Saving items to a cart that can then be emailed, printed, or exported to RefWorks
  • Search History

In addition, we are releasing the first library apps for smartphones and tablets. These library apps also support most modern web browsers on personal computers. To access the Tutt Library's growing number of apps, go to http://discovery.coloradocollege.edu/apps.

The new TIGER is an open-source project and you can contribute to its design and functionality. The source code repository is available on Github.
